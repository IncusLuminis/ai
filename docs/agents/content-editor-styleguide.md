# Content_Editor Style Guide

**Status:** Starter — empty by design. `Content_Editor` grows this from real drafts it reviews, rather than a rule set invented speculatively up front.

## How this file is meant to grow

Every time `Content_Editor` catches a real cliché, tic, or readability problem in a `Content_Master` draft, add it here as a rule with a short example — so the next review (by this role or a human) applies the same standard instead of relying on memory. Prefer "avoid X, do Y" pairs over abstract advice.
