# Content_Translator Style Guide

**Status:** Starter — empty by design. Grows from real translations, either logged by `Content_Translator` itself or written directly by a human reviewer (Mihal). Both are treated as binding guidance with equal weight - a human note here isn't a "suggestion," it's a rule the next translation pass must follow.

## How this file is meant to grow

Whenever a specific EN->RU rendering choice matters enough to standardize - a term that keeps getting translated inconsistently, a name/unit/notation convention, a phrase that reads wrong in Russian - add it here as a rule with a short example, the same way `Content_Editor` logs clichés in its own style guide. Prefer "avoid X, do Y" pairs over abstract advice.

## 1. Terminology / glossary

Specific English terms and their preferred Russian rendering, for consistency across translations of the same domain.

### Astronomy / planetary-science paper pipeline (EN->RU)

- C-complex asteroids -> C-комплексные (углеродистые) астероиды
- carbonaceous chondrites (CC) -> углистые (углеродистые) хондриты
- phyllosilicates -> филлосиликаты
- aqueous alteration -> водное изменение
- space weathering (SW) -> космическое выветривание
- band center / band depth (Bc / Bd) -> центр полосы / глубина полосы (сохранять латинские аббревиатуры Bc/Bd в скобках при первом упоминании)
- Euclidean distance -> евклидово расстояние
- Near-Earth Asteroids (NEA) -> околоземные астероиды (НОА); при первом упоминании давать англ. аббревиатуру NEA в скобках, т.к. в источниках и последующих этапах пайплайна её используют как есть
- Main Belt / Inner Main Belt (IMB) -> Главный пояс / Внутренний Главный пояс
- parent body -> родительское тело
- thermal model (e.g., NEATM) -> названия моделей (NEATM и т.п.) не переводятся, даётся расшифровка при первом упоминании
